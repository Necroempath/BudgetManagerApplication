The project provides possibility to communicate with data base by adding, removing and editing operations in the browser local storage. UI introduces an interactive table of operations, 
which user can sort by different parameters, such as title, amount, date. Filtering has realized through completely new form, poping up in canvas. Search by name input, 
dedicate to find desired operation among filtered operations. Application reacts sensetively on currency change, saving new choice in cookie, affects UI and call recalculations of totals, based on currency type.
Possibility of import and export Json is also introduced for making easier testing for developer as well as satisfy user needs. 
Now, app also provides authorization/registration page for corressponding functionality. User password going through hasher to provide secure data saving paradigm. Information about current user is keeping in cookies,
and user can remember self in order to entry without authorization in future.
